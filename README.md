# Cat Breeds Website

A colorful cat breeds website built with HTML, CSS, and JavaScript.

## Features

- Cute cat breed cards
- Login and sign-up form design
- Floating heart animation
- Responsive grid layout
- Soft pink and blue theme

## Files

- `index.html` — main website file
- `README.md` — project information

## How to View

Open `index.html` in a browser.

## GitHub Pages

To publish this project:

1. Upload the files to a GitHub repository.
2. Go to **Settings** → **Pages**.
3. Choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save and open the link GitHub gives you.
